## 🎯 信號生成準則

### 第一階段：趨勢判斷

**數據來源**: WebSocket實時K線（1h/15m/5m三時間框架）

**判斷標準**:
```
趨勢 = f(EMA20, EMA50, EMA斜率, ADX)

┌─────────────────────────────────────────────────┐
│ Bullish (看漲):                                  │
│   ✓ EMA20 > EMA50                               │
│   ✓ EMA20斜率 > 0.01                            │
│   ✓ ADX > 20                                    │
├─────────────────────────────────────────────────┤
│ Bearish (看跌):                                  │
│   ✓ EMA20 < EMA50                               │
│   ✓ EMA20斜率 < -0.01                           │
│   ✓ ADX > 20                                    │
├─────────────────────────────────────────────────┤
│ Neutral (震盪):                                  │
│   ✓ 不滿足以上任一條件                           │
│   ✓ 或 ADX < 20                                 │
└─────────────────────────────────────────────────┘
```

### 第四階段：信心度計算（ICT五維評分）

```
信心度 (0-100分) = 5個維度加權總和
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ EMA偏差評分 (40%)
   • H1 EMA20偏差率: 0-10分
   • H1 EMA50偏差率: 0-10分
   • M15 EMA20偏差率: 0-10分
   • M15 EMA50偏差率: 0-5分
   • M5 EMA20偏差率: 0-5分
   
   評分邏輯:
   ┌────────────────────────────────────┐
   │ 偏差率 < 0.5%: 滿分               │
   │ 偏差率 0.5-2%: 線性遞減           │
   │ 偏差率 > 2%: 0分                  │
   └────────────────────────────────────┘

2️⃣ 市場結構評分 (20%)
   • Bullish結構 + LONG信號: 20分
   • Bearish結構 + SHORT信號: 20分
   • Neutral結構: 10分
   • 對立結構: 0分

3️⃣ 價格位置評分 (20%)
   • 接近有效OrderBlock: +10-20分
     - 未測試過的OB: 20分
     - 測試1次的OB: 15分
     - 測試2次的OB: 10分
     - 測試3次+的OB: 5分
   
   • 接近流動性區域: +5-15分
     - 強度3+聚集: 15分
     - 強度2聚集: 10分
     - 強度1聚集: 5分

4️⃣ 動量評分 (10%)
   • RSI背離檢測: +5-10分
   • MACD確認: +0-5分

5️⃣ 波動性評分 (10%)
   • ATR適中 (0.015-0.03): 10分
   • ATR過高 (>0.03): 5分
   • ATR過低 (<0.015): 5分

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
最終信心度 = 總分 × ADX懲罰 ÷ 100

例如:
• 總分 = 75分
• ADX = 18 → 懲罰 = 0.8
• 最終信心度 = 75 × 0.8 ÷ 100 = 0.60 (60%)
```

### 第五階段：勝率計算（基於EMA偏差）

```
勝率 (0-1) = f(EMA偏差, 信心度, R:R, 市場結構)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

基礎勝率計算:
┌──────────────────────────────────────────────┐
│ 完美對齊 + 低偏差:                            │
│   基礎勝率 = 0.70                             │
│   調整範圍: 0.65-0.75                         │
├──────────────────────────────────────────────┤
│ 強趨勢 + 中偏差:                              │
│   基礎勝率 = 0.60                             │
│   調整範圍: 0.55-0.65                         │
├──────────────────────────────────────────────┤
│ 弱趨勢 + 高偏差:                              │
│   基礎勝率 = 0.50                             │
│   調整範圍: 0.45-0.55                         │
└──────────────────────────────────────────────┘

調整因子:
• 市場結構對齊: +5%
• R:R > 2.0: +3%
• 信心度 > 70%: +2%
• OrderBlock確認: +5%

最終勝率 = 基礎勝率 × (1 + 調整因子總和)
範圍限制: 0.35 ~ 0.85
```

### 第六階段：動態SL/TP計算

```
止損/止盈設置（基於ATR）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

基礎止損:
  SL_distance = ATR × 2.0

基礎止盈:
  TP_distance = SL_distance × R:R_ratio
  
  R:R範圍: 1.0 ~ 3.0
  • 高信心信號: R:R = 2.5-3.0
  • 中信心信號: R:R = 2.0-2.5
  • 低信心信號: R:R = 1.5-2.0

OrderBlock調整（如果價格接近OB）:
  • LONG信號接近支撐OB:
    SL = min(基礎SL, OB下方0.3%)
  
  • SHORT信號接近阻力OB:
    SL = max(基礎SL, OB上方0.3%)
```

• H1: bullish=80, bearish=70, neutral=250

↓

階段3: 信號方向確定 (400 → 30-120個)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 優先級1 (完美對齊): 0-10個
• 優先級2 (H1+M15): 5-20個
• 優先級3 (趨勢初期): 10-30個

寬鬆模式（RELAXED=true）:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

↓

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

計算: ATR動態 → SL/TP/R:R

↓

階段6: 雙門檻驗證 (18-78 → 10-50個)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 信心 ≥ 40%
✓ R:R ∈ [1.0, 3.0]

↓

階段8: 多信號競價排序 (8-40 → 前5個)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
排序: score = confidence×0.4 + win_prob×0.4 + rr×0.2

### 信心度計算範例

```
範例: BTCUSDT LONG信號
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• 流動性區: 64250 (支撐, 強度3)
• ADX: 22

計算過程:

1️⃣ EMA偏差評分 (40%):
   H1 EMA20偏差 = |64700-64500|/64700 = 0.31% → 10分
   H1 EMA50偏差 = |64700-64000|/64700 = 1.08% → 7分
   M15 EMA20偏差 = |64700-64600|/64700 = 0.15% → 10分
   M15 EMA50偏差 = |64700-64200|/64700 = 0.77% → 4分
   M5 EMA20偏差 = |64700-64650|/64700 = 0.08% → 5分

2️⃣ 市場結構評分 (20%):
   bullish結構 + LONG → 20分

3️⃣ 價格位置評分 (20%):
   接近OrderBlock (64300) → 20分 (未測試)
   接近流動性區 (64250) → 15分 (強度3)

4️⃣ 動量評分 (10%):
   RSI無背離 → 5分

5️⃣ 波動性評分 (10%):
   ATR = 0.024 (適中) → 10分

總分: 36 + 20 + 20 + 8 + 10 = 94分

最終信心度 = 94 / 100 = 0.94 (94%) ✅
```
